# WhaBot-AI-Companion

Interact seamlessly with OpenAI through WhatsApp, enabling prompt-based conversations with AI models.


## Steps for setting up the project

-> Install yarn
-> Install husky
-> Install eslint & eslint-staged
-> Install commitlint
